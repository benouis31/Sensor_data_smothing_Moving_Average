# Sensor_data_smothing_Moving_Average
Analyzes sensor data with moving average smoothing
## Features
- Automatic delimiter detection  
- Data cleaning and merging  
- Moving-average smoothing with:
  - 30-minute window  
  - 6-hour window  
  - 12-hour window  
- Side-by-side comparison of original vs smoothed signals  
- Statistical comparison tables
